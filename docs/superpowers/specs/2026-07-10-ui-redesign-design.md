# UI 리디자인 — MyRoutine

날짜: 2026-07-10
상태: 승인됨 (사용자가 방향 확정: 이름 MyRoutine, 무드 "클린 베이스 + 루틴 쪽 게임화 포인트")

## 목표

1. 앱 이름을 "예림의 루틴" → **MyRoutine**으로 변경
2. **데스크톱 반응형**: 넓은 화면에서 루틴/할 일 2단 배치 (지금은 480px 고정이라 양옆이 빔)
3. **루틴 탭 동기부여 강화**: 클린한 베이스 위에 게임화 포인트 (스트릭·달성률 연출)
4. **할 일 탭 정돈**: 날짜 표기 간결화, 시각 위계 정리

기능 변경 없음 — 순수 프레젠테이션 레이어 작업. 데이터 로직은 건드리지 않는다.

## 1. 브랜딩

- `index.html`: `<title>MyRoutine</title>`, `apple-mobile-web-app-title` → MyRoutine
- 헤더: 좌측에 `🌱 MyRoutine` 브랜드 (작게), 날짜 선택은 그 아래 유지
- `AuthScreen`: 제목 "루틴 & 할 일" → "MyRoutine"

## 2. 반응형 레이아웃

- `index.css`의 `#root { max-width: 480px }` 제거 — 폭 제어를 App으로 이관
- **모바일 (<1024px)**: 기존 그대로 — 단일 컬럼 `max-w-[480px]` 중앙, 탭 전환
- **데스크톱 (lg, 1024px+)**: `max-w-5xl` 중앙, **루틴 | 할 일 2단 그리드** 동시 표시,
  탭 바 숨김 (`lg:hidden`), 각 컬럼 상단에 섹션 타이틀
- 구현: 두 패널을 항상 렌더하고 모바일에서만 비활성 탭을 `hidden` 처리
  (두 탭 모두 마운트 시 데이터를 각자 로드하는 기존 구조 그대로 활용)
- 헤더는 컨테이너 폭 전체를 차지. 명언은 헤더에 현행 유지, 날짜 선택은
  데스크톱에서도 루틴 컬럼에만 영향 (TodoTab은 selectedDate를 받지 않음 — 의도된 동작)
- **바텀시트 모달 (RoutineCalendar, TodoArchive 편집)**: 데스크톱에서는
  중앙 다이얼로그로 전환 (`lg:items-center lg:justify-center` + 시트에
  `lg:max-w-md lg:rounded-3xl`). 모바일은 현행 바텀시트 유지
- `body` 배경색(`#f8f7ff`)을 앱 배경(`#f0fdf4`)과 통일 — 컨테이너 밖 투톤 방지

## 3. 루틴 탭 — 게임화 포인트

- **달성률 카드**: 프로그레스 바를 emerald 그라디언트로, 100% 달성 시
  카드가 축하 상태로 전환 (그라디언트 배경 + 🎉 + "완벽한 하루!")
- **스트릭 칩**: 활성 시 오렌지 그라디언트 pill (`🔥 3일` / `🔥 2주`),
  비활성 시 회색. 지금의 맨 이모지+숫자보다 존재감 있게
- **주간 진행도**: `이번 주 2/3` 텍스트 → **도트 ●●○ + 숫자** (목표 최대 6이라 도트 가능).
  목표 달성 시 도트 전체 emerald + 배지 강조. 목표 초과 시(주 3회에 4일 체크)
  도트는 목표 수까지만, 숫자는 `4/3` 그대로 표시
- **체크 애니메이션**: 체크 시 원이 살짝 튀는 scale 트랜지션 (CSS만, 라이브러리 없음)
- 카드 스타일은 현행 유지 (클린 베이스) — 게임화는 위 포인트에만 집중

## 4. 할 일 탭 — 정돈

- **날짜 표기 간결화**: `📅 2026-07-10` → 상대 표기 (`오늘`, `내일`, `어제`,
  그 외 `7/15`; 연도가 다르면 `2025.12.30`), `⏰ 2026-07-10` → D-day
  (`D-3`, `오늘까지`, `지남`). 기한 지남/임박(D-3 이내)은 red 계열
- 상대 표기 헬퍼는 `src/lib/dates.js`에 순수 함수로 추가 (`formatRelativeDate`, `formatDday`)
- **아카이브(완료된 투두)**: 날짜는 `formatRelativeDate`만 적용, D-day·경고색은
  쓰지 않는다 — 완료된 항목에 "지남" 빨간 표시는 잘못된 신호
- 섹션 헤더: 카운트를 배지 스타일로, 섹션 간 간격 정리
- 카드 내부 타이포/간격 미세 조정. 태그 칩은 현행 유지

## 5. 코드 정리 (이 작업에서 자연히 걸리는 것만)

- `App.css`: Vite 템플릿 잔재 (`.counter`, `.hero` 등 미사용 스타일) 삭제
- lint 기존 에러 정리:
  - `App.jsx` 미사용 변수 `today` 제거
  - `RoutineTab`의 `checks` state → `allChecks[dateKey]` 파생값으로 변경
    (set-state-in-effect 에러 해소 + 상태 중복 제거)
  - `App.jsx` 마이그레이션 이펙트의 setState는 일회성 게이트 특성상 유지,
    eslint-disable 주석 + 사유 명시
